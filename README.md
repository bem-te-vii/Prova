# Prova 
Para executar o projeto, organize os arquivos em uma pasta raiz contendo o `server.js` e uma subpasta chamada `public` com os arquivos de interface (`index.html`, `style.css`, `script.js`), abra o terminal nessa pasta raiz, instale o framework necessário com o comando `npm install express`, inicie o servidor digitando `node server.js` e, por fim, acesse `http://localhost:3000` no seu navegador para visualizar o monitoramento em tempo real.
